# Functions in C++

C++ functions have a return type, name, and parameter list.

## Basic Function

```cpp
int get_number() {
    return 42;
}
```

## String Type

The `std::string` type holds text:

```cpp
#include <string>

std::string get_text() {
    return "some text";
}
```

## Header Includes

Use `#include` to bring in standard library features:

```cpp
#include <string>    // std::string
#include <iostream>  // std::cout
#include <vector>    // std::vector
```
